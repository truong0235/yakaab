### yet another kahoot auto answer bot 

### usage:
- install nodejs, npm 
- clone this repo
- set up room pin, room id, accounts in .env.example file
- change file name .env.example to .env
- install dependency packages (run once only):
```
    npm install
```
- start this tool with command:
```
    npm start
```
